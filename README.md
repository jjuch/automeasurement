# automeasurement
An automatic measuring and logging protocol


## Install (Windows)
### Prerequisites
* git (in terminal)
* python (in terminal)
* virtualenv
* NI drivers

### Procedure
1. Create a folder
2. Open 'cmd' and navigate to the folder
3. Clone this repo in the folder
```
    >>> git clone https://github.ugent.be/DySC-SYSTeMS/automeasurement.git
```
4. Run the setup
```
    >>> cd automeasurement
    >>> setup.bat
```
5. Change config files
